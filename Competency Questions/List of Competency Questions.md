# Competency Questions

The competency questions define the information requirements that guided the development of the Automation Bias Ontology (ABO).

The questions are grouped into three categories:
- AI Model and System
- Environmental Context
- Agent Profile

| ID | Category | Competency Question |
|---|---|---|
| CQ1 | AI Model and System | What AI output was provided to the human operator as part of the human oversight process? |
| CQ2 | AI Model and System | What AI model produced this AI output associated with this human oversight process?  |
| CQ3 | AI Model and System | What were the performance metrics of the AI model when the human operator had the same output as the AI model during a human oversight process? |
| CQ4 | Environmental Context | What was the operator's workload at the time of this human oversight decision? |
| CQ5 | Environmental Context | What information artifacts were accessed by the human operator, and what interaction evidence was recorded to support this during this human oversight process? |
| CQ6 | Environmental Context | What was the complexity of the task during this human oversight process? |
| CQ7 | Agent Profile | Which human operators were associated with this task? |
| CQ8 | Agent Profile | What was the human oversight decision of the operator for this AI output? |
| CQ9 | Agent Profile | What was the operator's level of expertise during this human oversight process? |
| CQ10 | Agent Profile | What was the operator's AI literacy score during this human oversight process? |
| CQ11 | Agent Profile | What was the duration the operator spent on this human oversight process? |
